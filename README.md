# React Native Google Cast
A React Native library to use Google Cast

## Getting started

```
$ npm install rn-google-cast --save
```

### Installation

```
$ react-native link rn-google-cast
```
Notes: 
- RN 0.60+ can use Autolinking
- Only tested with RN 0.60+ <b>XD</b>
